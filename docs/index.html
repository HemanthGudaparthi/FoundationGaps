AI Explanations — bring your own key or run locally

Wikipedia and arXiv always work for free with no key. Add an API key to also get plain-English AI explanations. You can use OpenAI, point it at a local Ollama server (completely free, runs on your own machine), or any OpenAI-compatible endpoint.

Run 100% offline with Ollama

Install ollama.com, run ollama pull llama3.2, then select "Ollama (local)" above. Everything — video, keywords, AI explanations — stays on your machine. Free forever.

Best models for STEM & scientific topics

General models work, but these open-source models are significantly stronger at physics, math, biology, CS, and research literature:

phi-4

Microsoft — strong STEM reasoning, fast

qwen2.5-math

Alibaba — math & science specialist

deepseek-r1

DeepSeek — research-grade reasoning

llama3.1

Meta — general, reliable on STEM

Install any of these: ollama pull phi-4 → select "Ollama (local)" → type the model name above. All trained exclusively on open-access scientific literature.

Privacy

Your video, audio, and notes never leave your device. Only the text of clicked keywords is sent to the API you configure. Wikipedia and arXiv require no account. Keys live only in your browser's localStorage.
