Run 100% offline with Ollama

Install ollama.com, run ollama pull llama3.2, then select "Ollama (local)" above. Everything — video, keywords, AI explanations — stays on your machine. Free forever.

Privacy

Your video, audio, and notes never leave your device. Only the text of clicked keywords is sent to the API you configure. Wikipedia and arXiv require no account. Keys live only in your browser's localStorage.
